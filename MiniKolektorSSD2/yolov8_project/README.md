# YOLOv8 分类模型评估报告 — MiniKolektorSSD2 表面缺陷检测

## 📋 项目概述

使用 YOLOv8n-cls（nano 分类模型）对 MiniKolektorSSD2 数据集进行二分类：**abnormal（缺陷）** vs **normal（正常）**。

## 📁 数据集信息

| 项目 | 训练集 | 测试集 |
|------|--------|--------|
| abnormal（缺陷） | 796 | 382 |
| normal（正常） | 64,394 | 27,698 |
| **总计** | **65,190** | **28,080** |
| 类别比例 (normal:abnormal) | 80.9:1 | 72.5:1 |

- 图片尺寸：64×64 RGB PNG
- **类别严重不平衡**：正常样本约为缺陷样本的 **80 倍**
- 原始数据集中 normal 图片嵌套在子目录中，已通过符号链接展平为 YOLO 兼容格式

## 🔍 结果分析

### ⚠️ 核心问题：类别不平衡导致缺陷召回率低

1. **总体准确率 99.24% 具有误导性**：由于正常样本占比 98.6%，即使全部预测为 normal 也能达到 98.6% 准确率
2. **缺陷类召回率仅 50%**：一半的缺陷样本被误判为正常，在工业质检场景中不可接受
3. **缺陷类 F1-Score 64.14%**：精确率尚可（89.67%），但召回率拖了后腿
4. **正常类表现极好**（99.92%），代价是牺牲了缺陷检出能力

### 🛠 改进建议

1. **处理类别不平衡**：
   - 对缺陷样本过采样（Oversampling）或对正常样本欠采样
   - 使用加权损失函数，提高缺陷类权重
   - 对缺陷样本做更多数据增强

2. **评估指标优化**：
   - 以 F1-Score / AUPRC 为主要指标
   - 调整分类阈值，牺牲部分正常类准确率换取更高的缺陷召回率

3. **模型层面**：
   - 尝试更大模型（yolov8s-cls / yolov8m-cls）
   - 使用 Focal Loss 处理难样本
